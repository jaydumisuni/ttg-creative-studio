# TTG Creative Studio Product Standard

The product target is all three together:

```text
Photoshop power + Canva simplicity + Filmora motion
```

## Photoshop power

TTG Creative Studio must support serious layered editing:

- Direct canvas selection.
- Drag, resize and rotate handles.
- Layer stack.
- Editable properties.
- Mask/background tools.
- Advanced text effects.
- Glow, stroke, shadow, gradient and reflection controls.

## Canva simplicity

TTG Creative Studio must stay easy for normal users:

- Easy Mode first.
- Templates and brand kits.
- Asset browser.
- One-click presets.
- Safe defaults.
- Quick export.
- Clean, non-overwhelming UI.

## Filmora motion

TTG Creative Studio must support motion and video creation:

- Motion Mode.
- Timeline tracks.
- Keyframes.
- Playback preview.
- Motion presets.
- Frame export.
- Optional Video Export Pack for MP4.

## Modes

The UI should be organized around three main modes:

```text
Easy | Advanced | Motion
```

Easy Mode should feel like Canva.
Advanced Mode should feel like Photoshop.
Motion Mode should feel like Filmora.

## Release rule

A feature is not considered done unless it is:

1. Easy enough for a normal user.
2. Powerful enough for advanced editing.
3. Connected to image/video output where relevant.
4. Tested by script or workflow.
5. Not hidden only in backend code.
