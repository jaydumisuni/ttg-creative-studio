# TTG Creative Studio Capability Report

## Working

- Editable .ttgstudio project schema
- Layer/action engine
- 2D canvas preview renderer
- Timeline/keyframe interpolation
- PNG/frame/MP4 export wrapper
- Project validation
- Pack status reader
- Creative Studio PyQt widget
- Standalone Creative Studio launcher

## Partial

- Cinematic intro builder scaffold
- Render planner with future 3D/audio tasks
- Optional pack manifest system
- PyInstaller spec for Creative Studio
- CLI helper for project/export tests

## Planned

- True 3D renderer backend
- Full layer editing handles on canvas
- Advanced timeline UI
- Pack downloader with checksum verification in UI
- Production-quality cinematic intro render pack
- Hunter control API
- Real release URLs for optional packs

## Blocked / not release-ready

- Heavy packs still use placeholder release URLs.
- True 3D rendering is planned but not implemented.
- MP4 export currently uses the 2D frame renderer only.
- UI is alpha-level and not yet full Photoshop/Filmora replacement.
- Release test should wait until alpha tests pass on Windows.
