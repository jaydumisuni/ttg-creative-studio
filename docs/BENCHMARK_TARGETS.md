# Benchmark Targets

TTG Creative Studio should not benchmark only against itself.

Use stronger tools and projects as references, then attempt practical reproduction tasks. The gap between our output and the benchmark becomes the next engineering target.

## Rule

```text
Find a better tool
Study what makes it better
Try to reproduce one output/workflow
Record what failed
Turn the gap into implementation work
```

## Benchmark groups

### Design / Canvas / Layered Editing

- Penpot: collaborative open-source design tool benchmark for panels, layout, constraints, components and professional design workflow.
- tldraw: benchmark for direct canvas manipulation, low-friction drawing, selection, handles and editor feel.
- Excalidraw: benchmark for simple approachable canvas UI and fast diagram creation.
- CreatiPoster: benchmark for editable multi-layer graphic/poster generation from assets and prompts.

### Motion / Video / Timeline

- OpenShot: benchmark for approachable timeline, tracks, keyframes, FFmpeg export and user-friendly video editing.
- Kdenlive / Flowblade / Shotcut class: benchmark for serious multi-track editing, effects, trimming and export workflow.
- Olive: benchmark for deeper professional NLE architecture ideas such as node/compositing direction, color management and render pipeline ambition.
- Remotion / Motion Canvas class: benchmark for code-driven motion templates, repeatable video rendering and programmable visual systems.

### 3D / Cinematic Output

- Blender: benchmark for real 3D text, camera, lights, depth, particles and cinematic scene rendering.

## TTG benchmark tasks

1. Build a 9:16 ad from a ZIP of images.
2. Build a cinematic THETECHGUY intro still.
3. Build a motion preview/contact sheet.
4. Build a simple editable template with text, image, glow, gradient and CTA.
5. Build a timeline scene with keyframes and preview.
6. Export frames and, when the Video Export Pack is installed, MP4.
7. Visually compare the result against the benchmark class.

## Gap report fields

For every benchmark test, record:

- Benchmark target.
- Task attempted.
- What TTG Creative Studio produced.
- Visual proof artifact.
- What looked worse.
- What workflow felt worse.
- Missing engine feature.
- Missing UI feature.
- Next implementation target.

## Current strategic takeaway

TTG Creative Studio should use benchmarks to discover what to push next, but it should not copy their identity.

The goal remains:

```text
Photoshop power + Canva simplicity + Filmora motion + TTG worker intelligence
```
