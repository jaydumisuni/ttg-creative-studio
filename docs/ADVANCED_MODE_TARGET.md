# TTG Creative Studio Advanced Mode Target

The current app is an alpha shell. It must not be called release-ready until Advanced Mode can create work close to the supplied THETECHGUY cinematic reference: neon panels, ghost/logo hero, 3D-looking wordmark, depth, particles, camera motion, reflections and polished export.

## Product standard

Advanced Mode should feel familiar to users of Photoshop, Canva and Filmora:

- Layer stack with visible/lock/rename/group/reorder controls.
- Click-select, drag, resize handles, rotate handles, snap/grid/guides.
- Editable properties panel for transform, text, color, fill, effects and animation.
- Timeline with ruler, playhead, keyframe dots, easing and duration controls.
- Effects panel: glow, shadow, stroke, bevel, extrusion, blur, reflection, gradient, noise, particles.
- Asset browser: brand kit, logos, icons, templates, backgrounds, overlays, sounds.
- Template system: YouTube intro, banner, watermark, thumbnail, logo reveal, schematic, ISP poster.
- Export workflow: PNG, transparent PNG, GIF, frames, MP4, YouTube preset.

## Immediate target for THETECHGUY intro

The app must generate a project that looks closer to the reference:

- Dark cyber floor/background with perspective lines.
- Large 3D-style THETECHGUY wordmark with bevel/extrusion/glow/reflection.
- Ghost hero logo as a real image layer, not a missing placeholder.
- Floating glass service cards: SYSTEMS, TOOLS, ISP, WEB.
- Purple/cyan particles and light trails.
- Tagline and URL with controlled spacing and glow.
- Timeline presets that animate panels, wordmark, ghost, particles and camera feel.

## Do not rush to build

Builder/installer work only resumes after the app can create an acceptable still frame and motion preview from Advanced Mode.
