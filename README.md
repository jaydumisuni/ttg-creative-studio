\# TTG Creative Studio



Local-first THETECHGUY creative editor.



Current base:

\- Background remover

\- Image processing core

\- GIF/loading support

\- Desktop executable build scripts

\- Installer scripts



Target:

\- Layer-based editor

\- Timeline/keyframe motion editor

\- Brand kit system

\- Diagram/schematic tools

\- 3D logo/intro renderer

\- Hunter/AI control later



AI is not the foundation. The editor must work manually first.
